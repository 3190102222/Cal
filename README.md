# 下载

安装`git`

在命令行运行
```sh
git clone git@github.com:3190102222/Cal.git
```

即可获得讲义的 markdown 源文件

# 使用建议

**最推荐**使用 [Obsidian](https://obsidian.md) 打开，因为我们就是在 Obsidian 中编辑的，并且会用到一些 Obsidian 特有的语法。
同时，使用 Obsidian 导出为 pdf 也十分方便。

也可以使用支持markdown语法的其他编辑器打开，如
- vscode
- typora
- HbuilderX
